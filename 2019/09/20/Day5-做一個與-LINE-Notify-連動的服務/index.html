建立一個 views/ 的資料夾，並建立兩個檔案

  • notify_index.html

    這邊需要修改 client_id 以及 redirect_uri
    這個檔案主要功能是讓使用者有個可以按按鈕觸發的地方，當然這只是個範例，實際上線可能就會直接 call API 導向 LINE Notify 頁面等等

  • notify_confirm.html

    修改 ajax 裡的 url，將網址換成之前部署的網址，若忘了可以在部署一次 sls deploy
